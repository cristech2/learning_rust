## Guessing Game for Rust

this is a simple command-line guessing game implemented in Rust. The code and instructions for this game can be found in the [Rust Book](https://book.rustlang-es.org/ch02-00-guessing-game-tutorial)

### How to Play

1. Program generates a random number between 1 and 100.
2. You have to guess the number.
3. After each guess, the program will tell you if your guess is too low, too high, or correct.
4. Keep guessing until you find the correct number!
5. Enjoy the game!.

### Principies used in this project:
- Handling User Input with `std::io`
- Generating Random Numbers with the `rand` crate
- Using Control Flow with `match` Statements
- Error Handling with `Result`
- Looping with `loop`
- Comparing Values with `cmp` Method
- Using External Crates in Rust
- Type Conversion and Parsing
- String Manipulation
- Basic Program Structure in Rust
- Debugging and Testing Rust Programs
- Building Command-Line Applications

### Requirements

- Rust installed on your machine. You can download it from [here](https://www.rust-lang.org/tools/install).
- Basic knowledge of Rust programming language.
- A terminal or command prompt to run the game.
- An internet connection to download the `rand` crate.

### Running the Game

1. Clone the repository or download the source code.
2. Navigate to the project directory in your terminal.
3. Run the following command to build and run the game:
4. ```bash
   cargo run
   ```
5. Follow the on-screen instructions to play the game.
